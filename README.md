## 🎯 Project Overview
An interactive data analytics solution built in Power BI to monitor employee attrition, track workforce demographics, and analyze job satisfaction levels. This dashboard helps HR teams uncover trends and implement data-driven retention strategies.

## 🛠️ Tech Stack & Skills
* **Visualization Tool:** Power BI Desktop
* **Features Used:** Data Modeling, DAX Measures, Cross-Filtering, Custom Visual Layouts

## 📈 Key Insights Tracked
* **Workforce Demographics:** Monitored a total workforce of **1,470 employees** with an average age of **37** and an average tenure of **7 years**.
* **Attrition Metrics:** Isolated **237 total attritions**, revealing a company-wide **attrition rate of 16.1%**.
* **Segment Breakdown:** Visualized attrition factors across Education Fields (highest in Life Sciences at 38%), age bands (dominant in the 26-35 range), monthly salary tiers, and specific job roles like Laboratory Technicians.
* **Satisfaction Matrix:** Built a granular job satisfaction heat matrix mapping satisfaction scores (1-4) across various organizational roles.
